## 常用网站导航

| 网站 | 用途 |
| --- | --- |
| [CERN](https://home.cern/) | 欧洲核子研究中心官网，适合了解大型实验、新闻和科普资料。 |
| [PDG](https://pdg.lbl.gov/) | Particle Data Group，查询粒子性质、综述和数据表。 |
| [arXiv hep-ph](https://arxiv.org/archive/hep-ph) | 高能物理现象学预印本入口。 |
| [INSPIRE HEP](https://inspirehep.net/) | 高能物理论文、作者、引用和实验信息检索。 |
| [HEPData](https://www.hepdata.net/) | 高能物理实验数据发布与查询平台。 |
| [BESIII](http://bes3.ihep.ac.cn/) | BESIII 相关实验信息与分析资料入口。 |
| [AI@IHEP](https://ai.ihep.ac.cn/) | 高能所 AI 平台与相关工具入口。 |
| [INK](https://ink.ihep.ac.cn/panel) | INK 面板入口，可用于站内服务与资源访问。 |
